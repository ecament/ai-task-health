# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-11

### Added
- Initial release of VM Health Check script
- CPU usage monitoring
- Memory usage tracking
- Disk space analysis
- System uptime reporting
- Load average monitoring
- Color-coded status indicators (Green/Yellow/Red)
- Detailed explanation mode with `explain` argument
- Professional formatted output with borders and sections
- Support for multiple Linux distributions
- Comprehensive README with usage examples
- Installation instructions and troubleshooting guide
- Cron scheduling examples
- Project structure with docs folder

### Features
- Real-time health parameter monitoring
- Educational explanations for each metric
- Automatic status calculation based on thresholds
- Cross-platform Linux compatibility
- Lightweight implementation using only standard utilities
